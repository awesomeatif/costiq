---

## What CostIQ Does

CostIQ analyzes hospital procurement and inventory data to surface:

- Vendor price inconsistencies  
- Contract vs invoice mismatches  
- Overstocking and expiry risk  
- High-cost purchasing patterns  

The output is a **clear, executive-ready diagnostic report** that helps finance and operations teams prioritize savings actions.

---

## How It Works

1. Hospitals export procurement and inventory data as CSV files  
2. CostIQ normalizes and analyzes the data using rule-based logic  
3. Findings are summarized in a CFO-ready PDF diagnostic
